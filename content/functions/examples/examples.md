> Functions you can use in your styles.

## Generating colors

```less
rgb(r, g, b)
```

```less
rgba(r, g, b, a)
```

```less
hsl(h, s, l)
```

```less
hsla(h, s, l, a)
```

## Modifying colors

```less
saturate(color, amount)
```

```less
desaturate(color, amount)
```

```less
lighten(color, amount)
```

```less
darken(color, amount)
```

```less
greyscale(color, amount)
```
